Documentation API

LiteAI est une API de chat IA universelle, compatible OpenAI et Anthropic.

🔑 Authentification

Toutes les routes API requièrent une clé valide via l'un de ces headers :

X-API-Key: sk-liteai-xxxxx
Authorization: Bearer sk-liteai-xxxxx

📡 Endpoints

POST /api/chat Chat complet (non-streaming) 
{
  "message": "Votre question",
  "model": "qwen3-32b",        // optionnel
  "session_id": "uuid",        // optionnel, pour continuer
  "max_tokens": 2048,          // optionnel
  "temperature": 0.7           // optionnel
}
POST /api/chat/stream Streaming SSE (temps réel)

Même format que /api/chat. Réponse en flux data: {"content": "..."}

POST /v1/chat/completions Compatible OpenAI 
{
  "model": "Qwen/Qwen3-32B",   // ou la clé: "qwen3-32b"
  "messages": [
    {"role": "user", "content": "Bonjour"}
  ]
}
POST /v1/messages Compatible Anthropic (Claude Code) 
{
  "model": "claude-3-5-sonnet-20240620",  // mappé vers Qwen3-32B
  "max_tokens": 1024,
  "messages": [{"role": "user", "content": "Bonjour"}]
}
GET /v1/models   |   /api/models Lister les modèles
POST /api/upload Uploader un fichier (txt, md, pdf, csv, xlsx)

Form-data avec champ file. Retourne le contenu extrait du fichier.

POST /api/export/{fmt} Exporter une réponse (txt, md, docx, xlsx, pdf) 
{
  "content": "Texte à exporter…",
  "filename": "mon_rapport"    // optionnel
}

🖥️ Exemples CLI

Utilisez LiteAI avec curl, llm, Claude Code, ou tout outil compatible OpenAI.

curl — Chat simple

curl -X POST http://localhost:5000/v1/chat/completions \
  -H "Authorization: Bearer sk-liteai-12345" \
  -H "Content-Type: application/json" \
  -d '{"model":"qwen3-32b","messages":[{"role":"user","content":"Bonjour"}]}'

Upload d'un fichier PDF

curl -X POST http://localhost:5000/api/upload \
  -H "X-API-Key: sk-liteai-12345" \
  -F "file=@document.pdf"

Export en PDF

curl -X POST http://localhost:5000/api/export/pdf \
  -H "X-API-Key: sk-liteai-12345" \
  -H "Content-Type: application/json" \
  -d '{"content":"Mon rapport...","filename":"rapport"}' \
  --output rapport.pdf

Configuration Claude Code

ANTHROPIC_BASE_URL=http://localhost:5000
ANTHROPIC_API_KEY=sk-liteai-12345
claude --model claude-3-5-sonnet-20240620
